# Ingress

Ingress는 외부의 http/https 요청을 서비스에 전달하는 정책을 정의합니다.\
특정 api 주소로 http 요청이 들어왔을 때, 정의한 라우팅 규칙에 따라 적절한 서비스로 요청을 포워딩합니다.\
nginx 등의 controller를 이용하여 작동합니다.
