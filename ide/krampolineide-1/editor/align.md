# 정렬

에디터 내용을 줄 단위로 정렬합니다.\
\
정렬하기 위해서는 **\[편집] > \[오름차순/내림차순 정렬]**로 가거나 기본 단축키 **`F9`**를 누릅니다. \
실행하면 에디터의 내용이 문자와 알파벳 순으로 정렬됩니다.\
\
오름차순/내림차순으로 한 번 정렬된 문서에서 한 번 더 정렬을 실행하면 반대 순서로 정렬됩니다.
