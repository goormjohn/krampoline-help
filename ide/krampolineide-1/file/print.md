# 인쇄

현재 활성화된 에디터의 내용을 인쇄합니다. 소스 코드에 하이라이팅이 입혀져 있을 경우 하이라이팅이 그대로 인쇄됩니다. 하이라이팅을 보시려면 컬러로 인쇄하세요.

파일을 인쇄하기 위해서는 **\[파일] > \[인쇄]**로 가거나 기본 단축키 **`Ctrl + P (Mac: ⌘P)`**를 누릅니다. \
그러면 현재 활성화된 에디터의 내용이 새 창으로 뜨게 되고 브라우저의 인쇄 창에서 인쇄를 진행합니다.
